# AGENTS.md

## Project

- **What**: どうぶつしょうぎ (Doubutsu Shogi / Animal Shogi) — a two-player local hotseat board game
- **Deploy**: GitHub Pages at `https://iszk1215.github.io/animalshogi/`
- **Pages root**: `docs/` directory (committed to repo, not generated)

## Commands

```
make              # Rebuild docs/output.css from input.css
npm install       # Install Tailwind v4 deps
```

No test, lint, or typecheck scripts exist.

## Architecture

- `docs/board.html` — game entry point; imports `index.js` as ES module and calls `init()`
- `docs/index.html` — landing page with link to `board.html`
- `docs/index.js` — all game logic in vanilla JS (ES module, ~570 lines): `Game`, `Board`, `Cell`, `Bench` classes
- `docs/output.css` — **build artifact**, generated by Tailwind CLI. Do not edit manually.
- `input.css` — Tailwind v4 entry (`@import "tailwindcss"`)
- `docs/img/` — piece images (lion, elephant, giraffe, chick, chicken)

## Key facts

- Board is 3x4 cells; uses integer positions `[x, y]` with `x in 0..2`, `y in 0..3`
- `index.js` exports a single `init()` function — no other public API
- Tailwind v4 CLI (`@tailwindcss/cli`) scans `docs/*.html` and `docs/*.js` for class usage
- No CI, no pre-commit hooks, no formatter config
